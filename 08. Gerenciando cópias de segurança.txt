Priori: O BACKUP do nosso BD é chamado de DUMP.

1. Vá até o PGAdmin4, autentique e clique com o botão direito no DB > Backup

2. Nomeie o filename

3. Formato: CUSTOM

4. Encoding: UTF-8

5. Role name: Postgres

> Backup;

Voltando Backup

1. Query Tool (botão direito)
DROP TABLE cursos;
DROP TABLE gafanhotos;
SELECT * FROM cursos;
SELECT * FROM gafanhotos;

2. Botão direito > Restore...

3. Formato Custom

4. Selecione o arquivo de dump

5. Role name: Postgres

6. Query Tool
SELECT * FROM cursos;
SELECT * FROM gafanhotos;
